# Painel de Ferramentas do Instituto de Imunogenética (Igen/AFIP)

Painel interativo desenvolvido para **centralizar e organizar as principais ferramentas utilizadas no Instituto de Imunogenética (IGEN)**.  
O sistema reúne tanto **ferramentas criadas pelo próprio instituto** quanto **ferramentas externas** que fazem parte da rotina de trabalho, facilitando o acesso e a navegação em um único ambiente visual e intuitivo.

<img width="1430" height="746" alt="1painel" src="https://github.com/user-attachments/assets/7af97bee-82bb-4e17-8e8c-dee26208e362" />

## Sobre o projeto

O painel foi construído com **HTML**, **CSS** e **JavaScript**, priorizando um design limpo, responsivo e de fácil interação.  
Seu objetivo é **otimizar o fluxo de acesso** às aplicações utilizadas pelas equipes do IGEN, agrupando-as em uma interface centralizada e acessível tanto em computadores quanto em dispositivos móveis.

Entre os principais destaques estão:
- Organização visual por categorias de ferramentas (internas e externas);
- Interface moderna e responsiva;
- Acesso direto a links úteis e aplicações de rotina.

<img width="207" height="448" alt="2mobile" src="https://github.com/user-attachments/assets/46eec008-9876-4eff-a58d-89ec8d480e93" />


## Estrutura e Navegação

O painel é dividido em seções que representam as diferentes categorias de ferramentas, apresentadas através de **botões e cards interativos**.  
Cada elemento possui uma breve descrição e um link que leva diretamente à aplicação correspondente.

<img width="1401" height="631" alt="3 botoes" src="https://github.com/user-attachments/assets/0b27aa86-99a6-4c2d-9687-ff5b8e2600d3" />


## Cards e Ferramentas

Os cards são links clicáveis que direcionam o usuário diretamente para cada ferramenta.
Cada card apresenta informações essenciais, como nome, descrição resumida e logotipo, organizadas de forma visual e intuitiva.

<img width="1396" height="521" alt="4card" src="https://github.com/user-attachments/assets/b56357fb-e8ee-4733-a9e1-ed4a3a6960e2" />

## Menus Suspensos

O painel conta com **menus suspensos** que organizam as ferramentas em categorias, facilitando a navegação e tornando a interface mais intuitiva.  
Ao clicar em cada menu, são exibidas as ferramentas correspondentes, permitindo ao usuário **explorar os recursos de forma rápida e visualmente organizada**.

<img width="343" height="377" alt="7menu" src="https://github.com/user-attachments/assets/28eda84e-9762-48a7-9e9c-0cbf080b213b" />


## Ferramenta de Busca

O painel conta com uma barra de busca que permite localizar rapidamente qualquer ferramenta pelo nome.
A busca é dinâmica, filtrando os cards conforme o usuário digita, tornando o acesso mais ágil e intuitivo.

<img width="962" height="566" alt="5busca" src="https://github.com/user-attachments/assets/e3af8b1b-9765-40ff-9502-ea9cf4b13715" />


## Logotipos do Cabeçalho

O painel apresenta os **logotipos do IGEN e da AFIP** no cabeçalho, reforçando a identidade institucional do projeto.  
Esses logotipos são **clicáveis** e direcionam o usuário para os **sites oficiais** dos respectivos laboratórios, facilitando o acesso rápido às páginas institucionais.

<img width="1433" height="89" alt="6logos" src="https://github.com/user-attachments/assets/5e9ba106-2fbc-4a95-9c10-cbe4b89d6385" />


## Tecnologias utilizadas

- **HTML5** – Estrutura e conteúdo  
- **CSS3** – Estilização e layout responsivo  
- **JavaScript** – Interatividade e manipulação dinâmica dos elementos
